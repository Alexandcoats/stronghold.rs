# Generate an Ed25519 Key Pair

## Run the Example

You can generate an example ED25519 Key Pair inside an ephemeral vault by running the following command from the
[client crate](https://github.com/iotaledger/stronghold.rs/tree/dev/client).

```bash
cargo run --example cli generate-key --key-type Ed25519 --vault-path "vault_path" --record-path "record_path"
```
