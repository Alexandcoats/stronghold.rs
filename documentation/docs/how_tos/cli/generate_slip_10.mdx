# Generate a SLIP10 Master Key

## Run the Example

You can generate a new [SLIP10](https://github.com/satoshilabs/slips/blob/master/slip-0010.md) master key with optional
size by running the following command from within the
[client crate](https://github.com/iotaledger/stronghold.rs/tree/dev/client).

```bash
cargo run --example cli slip10-generate --size 512 --vault-path "slip10-vault-path" --record-path "slip10-record-path"
```
